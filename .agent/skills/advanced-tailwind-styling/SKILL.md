# Native Visual Styling Guidelines

## Goals

- Feel like a polished macOS utility, not a transplanted website
- Prioritize legibility and trust
- Use depth sparingly

## Guidance

- Use clear type hierarchy over oversized decorative treatments.
- Prefer subtle materials and spacing to fake glass overload.
- Keep menu bar surfaces compact and information-dense.
- Use status color carefully and always with text labels.

## Avoid

- web hero-section aesthetics
- dense gradients that compete with battery information
- styling that hides unsupported or unknown states
